# E-Commerce-Sales-Insights-Dashboard-Paxx-Store
Developed an interactive Excel dashboard to analyze Paxx Store’s sales data across leading e-commerce platforms. The project highlights key metrics including revenue trends, customer behavior, and channel performance. Advanced Excel features like slicers, pivot charts, and dynamic filtering enable real-time data exploration and decision-making.
